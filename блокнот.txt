функциональный модуль в С# приложение состоит из классов которые содержат
именнованные блоки кода называемыми методами 
метод (функции это часть класса, который может осуществлять действие или вычислять вычисления в использовании методов 
метод это наборы операторов которые выполняются вместе для создания метода необходимо создать его имя определить
способ  и тело метода для вызова метода другого класса необходимо, что бы он
был обявлен как паблик ызов осуществляется по имени класса и метода 
если метод не воид, то необходимо вернуть значение соответствующего типа оператор ретюрн
останавливает метод выполнения и передает управления вызвавшему 
данный метод оператору 
каждый метод имеет набор своих локальных переменных они видны только в нем и при завершении работы метода уничтожаются
для того чтобы переменные были видны в нескольких методов класса, необходимо обявитьполями не метода но внутри класса
 для невоид методов необходимо возвращать значение каждый "путь выполнения метода должен заканчиваться оператором ретурн"
для воид методов опреатора необязателен 
Использование параметров параметры позволяют передавать из одного метода в другой при обялении метода можно создать список его параметров 
если список пустой, то это означает что метод не имеет параметров.
С# существует 3 варианта передачи параметров: 
По значению (изменение значения параметра в методе не влияет в вызвавшем методе) 
По ссылке (Необходимо указывать ключевое слово рев до вызова метода необходимо обязательно инициализироать меременные в С#)
выходные параметры
